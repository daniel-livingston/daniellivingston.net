ABOUT

Hi, I’m Daniel Livingston. I’m a recent graduate with a master’s degree in computer science, and I’m ready to join your team!

Thanks to my education, I have a lot of knowledge for a junior developer, and I’m always trying to learn more. In 2020 alone, I expanded my knowledge to include at least 20 technologies that I had no prior knowledge of, including Typescript and SCSS. This flexibility allows me to quickly adapt to new development environments and learn new skills effectively. I’ve put these skills into practice, creating solid web applications that showcase my talents. I prioritize responsive web applications that look great in both a mobile and a desktop environment. I’m more of a developer than a designer, but I believe in making the user experience as clean and refined as possible. You can check out some examples of my work below.

If you’re interested in having me join your team, or even if you just have some questions and want to chat, let me know! I’m always open to meeting new people.

Front End

Focused on great, responsive designs with a mobile-first mentality. I tend to prefer minimalism in design but I definitely enjoy also making dynamic, expressive web pages. I’m experienced working with React as a framework, and I’ve explored other frameworks like Vue and Angular lightly, though I most appreciate React’s design philosophy.

Skills
  • React with React-Router, Redux
  • Jest
  • Sass/SCSS
  • Bootstrap
  • Solid HTML/CSS/JavaScript fundamentals

Back End

I prioritize creating thoughtful, well-designed, secure, and well-tested RESTful APIs for integration with front-end applications. Most of my experience is with the Express framework for Node.js, with which I have created apps that integrate with databases (both SQL and noSQL), implement user authentication, allow websocket communication.

Skills
  • Node.js
  • Express with common middleware, like body-parser and express-session
  • Testing with supertest and jest
  • User authentication with Passport and custom JWT-based methods
  • MongoDB and PostgreSQL
  • RESTful API design and implementation
  • Template engines like Pug and Handlebars
